Instalación de MySQL bajo Linux

  • Descargad los ficheros RPM de la página de recursos (Linux base y cliente).
  • Cambiar a usuario root.
  • Instalarlos con rpm -i nombre_del_fichero (primero el base, después el cliente)
  • Poner la password de root: /usr/bin/mysqladmin -u root password 'password'
  • Para acceder a MySQL: mysql -u nombre_usuario -p nombreBD
  • Los usuarios de MySQL no son usuarios del sistema, por lo que debemos crear usuarios. Para crear un usuario ejecutamos la siguiente sentencia SQL:
    • GRANT ALL PRIVILEGES ON *.* TO admin@localhost IDENTIFIED BY 'contraseña' WITH GRANT OPTION;
  • Cada vez que apagamos Linux el demonio deja de funcionar. Para poner en funcionamiento el servidor: safe_mysqld&

Instalación de PostGres bajo Linux

  • Descargad el fichero de PostGres para Linux
  • Descomprimirlo en un directorio cualquiera: tar -xvzf postgres...
  • Cambiar al directorio generado y teclear:
    • ./configure
    • gmake
    • su
    • gmake install
    • adduser postgres
    • mkdir /usr/local/pgsql/data
    • chown postgres /usr/local/pgsql/data
    • chgrp postgres /usr/local/pgsql/data
    • su - postgres
    • /usr/local/pgsql/bin/initdb -D /usr/local/pgsql/data
    • /usr/local/pgsql/bin/postmaster -i -D /usr/local/pgsql/data &
    • /usr/local/pgsql/bin/createdb nombre_BD
    • /usr/local/pgsql/bin/psql nombre_BD
  • Los usuarios de PostGres son usuarios de Linux. Para crear un usuario en Linux siendo root: adduser usuario. Para crear un usuario en PostGres /usr/local/pgsql/bin/createuser usuario
  • Cada vez que arranquemos Linux debemos ejecutar:
    • /usr/local/pgsql/bin/postmaster -i -D /usr/local/pgsql/data &
      para poner en marcha el servidor
