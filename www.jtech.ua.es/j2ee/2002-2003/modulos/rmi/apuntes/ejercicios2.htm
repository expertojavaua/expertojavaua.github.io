    • Añadir un servicio remoto para que un usuario se de de baja de la lista de participantes, añadiendo el método remoto:
      public void unregister(Chat c,String name) throws RemoteException;
    • Añadir un servicio remoto para que un usuario pueda registrarse como usuario VIP:
      public void registerVIP(Chat c,String name) throws RemoteException;
    • Añadir un servicio remoto para que un usuario pueda mandar un mensaje solamente a los usuarios VIP:
      public void postMessageVIP(Message m) throws RemoteException;
    • Añadir un servicio remoto para que un usuario pueda darse de baja como usuario VIP:
      public void unregisterVIP(Chat c,String name) throws RemoteException;
