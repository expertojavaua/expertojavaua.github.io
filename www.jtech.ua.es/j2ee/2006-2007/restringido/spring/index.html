Presentación

Spring es un framework cuya filosofía de desarrollo de aplicaciones J2EE es totalmente opuesta a la propugnada por Sun: donde el estándar usa sofisticados servidores de aplicaciones para dar diferentes servicios a nuestros complejos objetos de la capa de negocio (EJBs), Spring pretende conseguir lo mismo pero con servidores web y objetos Java convencionales. Lo que empezó como una filosofía "alternativa" ha ido creciendo hasta convertirse en uno de los frameworks J2EE más completos.
