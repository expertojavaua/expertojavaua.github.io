Presentación

Este módulo es una introducción a las aplicaciones web, y al uso de servidores que les den soporte. Se verá la estructura y configuración que debe seguir cualquier aplicación web JavaEE, y cómo configurar un servidor web (Tomcat) para alojar dichas aplicaciones.
