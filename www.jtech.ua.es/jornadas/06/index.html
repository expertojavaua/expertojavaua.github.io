I Jornada de Tecnologías Java

Bienvenidos a la I Jornada de Tecnologías Java, organizada por el Título de Especialista Universitario en Java Enterprise de la Universidad de Alicante. La jornada se celebrará el viernes día 12 de Mayo en el Salón de Actos de la Escuela de Óptica de la Universidad de Alicante. La inscripción es gratuita y está abierta a todos aquellos que tengan curiosidad en conocer las tecnologías Java.

Hemos preparado un gran día de conferencias y actividades para introducirte en el apasionante mundo de las tecnologías Java. Tendremos oportunidad de conocer en profundidad este lenguaje de programación, así como enfoques, herramientas y frameworks con los que resolver nuestros problemas informáticos de una manera profesional, robusta y abierta. Lo haremos de la mano de expertos ingenieros que utilizan diariamente estas tecnologías para el diseño y desarrollo de complejas aplicaciones comerciales. Ven a conocer Java el viernes 12 de Mayo a la Universidad de Alicante.

Horario y conferencias

Sesión de mañana (de 9:30 a 14:00 horas)

  • 9:30-10:00 h.: Registro y recogida de documentación.
  • 10:00-11:00 h.: Tecnologías Java para la Web: un enfoque práctico.
    Introducción a distintas tecnologías Java para el desarrollo de aplicaciones Web (Tomcat, JSP, JSF, AJAX, Struts, Spring, OJB), así como del uso de patrones de diseño orientados a J2EE. Con ejemplos prácticos de uso de estas tecnologías en aplicaciones para la gestión y edición de prensa on-line. Por José Galiana de Recursos en la Red.
  • 11:00-11:45 h.: Frameworks para persistencia en Java.
    Repaso de los distintos frameworks para persistencia. Desde el acceso directo a la base de datos con JDBC hasta Hibernate3, pasando por iBatis, JDO y EJB 2. Por Rubén Inoto, Ingeniero Informático.
  • 11:45-12:15 h.: Pausa.
  • 12:15-13:00 h.: Arquitecturas de alta disponibilidad y escalabilidad en Java.
    Algunas aplicaciones, por la criticidad del proceso de negocio que resuelven, tienen unas características especiales en cuanto a rendimiento, escalabilidad, fiabilidad y alta disponibilidad. Los servidores de aplicaciones Java permiten distintas topologías y configuraciones que podemos adaptar a la problemática de negocio de nuestros clientes. Por Ferrán García de Oracle.
  • 13:00-13:45 h.: Java y la profesión informática.
    La profesión informática desde la perspectiva de antiguos alumnos de la Universidad de Alicante con una extensa experiencia profesional. Por Aitor Medrano y Rubén Inoto, Ingenieros Informáticos.

Sesión de tarde (de 16:00 a 18:30 horas)

  • 16:00-16:45 h.: Web 2.0, un mundo de nuevas posibilidades.
    Una nueva forma de hacer la Web, más participativa y colaborativa. Los blogs, la sindicación de noticas o los agregadores de contenidos prometen cambiar radicalmente Internet. Por Pablo Vázquez de IT Deusto.
  • 16:45-17:30 h.: Introduciéndose en el mundo de Java EE.
    Introducción a Java Enterprise Edition. Consejos sobre entornos de desarollo a usar, servidores de aplicaciones, tendencias, etc. Por Martín Pérez de IT Deusto.
  • 17:30-18:30 h.: Mesa redonda sobre Java y la profesión informática.
    Participan: Manuel Cazorla (Director General de Sistel), Ferrán García (Oracle), Rubén Inoto (Ingeniero Informático) y Pablo Vázquez (IT Deusto). Moderada por Domingo Gallardo (Director del Titulo de Especialista Universitario en Java Enterprise).
  • 18:30 h.: Presentación del Java User Group de la Universidad de Alicante.
    Presentación del grupo de usuarios Java recién creado en la Universidad de Alicante. Sorteo de un iPod nano entre los asistentes a la jornada. Por Domingo Gallardo, del Especialista Universitario en Java Enterprise.
